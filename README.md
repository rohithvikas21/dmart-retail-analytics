# DMart Retail Sales Analytics & Behavioral Insights

## 📊 Executive Project Overview
This repository contains an end-to-end data analytics architecture designed to evaluate corporate retail transactional performance. Utilizing a large-scale, high-dimensional dataset containing **50,000 unique customer transactions** across 17 columns (encompassing operational metrics, GST tracking, gross revenue, and customer behavior data), this project models true enterprise retail strategy.
